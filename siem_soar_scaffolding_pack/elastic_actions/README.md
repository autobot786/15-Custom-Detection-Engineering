# Elastic Security – Rule Actions + Webhook Payload Examples

This folder contains:
- `rule_ndjson_examples.ndjson` – Example detection rules (export-style) with actions placeholders
- `webhook_payload_examples.json` – Example payloads for webhook connectors
- `actions_guidance.md` – How to wire actions in Kibana (conceptual)

> You must update connector IDs and indexes for your environment.
